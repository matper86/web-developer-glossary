Web Developer Glossary

Emmet

Es una colección de plug-ins para editores de texto, que realizan diferentes tareas que facilitan la edición de páginas web.

Markdwon

Es un lenguaje simple que se utiliza para crear texto enriquecido, para editar archivo .md, por ejemplo.

VSCode

Es un editor de codigo.

Git

Es un gestor de versionado para nueatro software, que se usa con Gitbash.

CodeSandbox

Editor de codigo al estilo de VSCode pero online.

Node.js®

Es un entorno de ejecución para JavaScript construido con V8, motor de JavaScript de Chrome.

NVM

Gestor de versiones instaladas de Nodejs.
